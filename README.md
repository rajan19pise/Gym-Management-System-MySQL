Gym Management System (MySQL Project)

This project is a relational database system designed for managing gym operations.

Features
- Member management
- Trainer management
- Membership plans
- Payments tracking
- Attendance system
- Workout plans

Database Design
The system uses multiple related tables with foreign key relationships:
- Members
- Trainers
- Membership_Plans
- Payments
- Attendance
- Workout_Plans

How to Run
1. Open MySQL Workbench or CLI
2. Run gym_database.sql
3. Database will be created automatically
4. Inserted sample data will be loaded

Tech Used
- MySQL
